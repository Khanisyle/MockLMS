Mock LMS Project

"Introduction to Web Design" on Google Classroom

By Khanisyle

In this project, I created a mock course on Google Classroom, designed to teach basic web design concepts. The course covers HTML basics with course material, interactive quizzes, and progress tracking.

Key Features:

  • Course Structure: Organized into one module, with resources and quizzes.
  • Quizzes: Created an interactive Google Form quiz to test student knowledge.
  • Learner Progress Tracking: Monitored student progress through the quiz providing feedback for improvement.
  • Customization: Designed a custom theme for the course and included external resources like Google Docs.

Technologies Used:

  • Google Classroom
  • Google Forms (for quizzes)
